# Обзор

Это упражнение предполагает модификацию скрипта MATLAB, реализующего компактную схему решения уравнения диффузии. Задача состоит в том, чтобы включить внешнюю силу, сравнить численное решение с истинным решением и проанализировать точность для различных размеров сетки.

1. **Учесть внешнюю силу**: Усовершенствовать код MATLAB для учета внешней силы $f(t, x)$.

## Коэффициенты вычислительных схем

В этом разделе описаны коэффициенты, используемые в трех вычислительных схемах: явной схеме Эйлера, схеме Кранка-Никольсона и компактной схеме. Эти коэффициенты имеют решающее значение для устойчивости и точности численных решений дифференциальных уравнений.

### 1. Явная схема Эйлера (Euler\_ex)

Явная схема Эйлера - это прямой метод интегрирования по времени, используемый для решения обыкновенных дифференциальных уравнений. Это метод первого порядка.

#### Коэффициенты:

* Левая часть:
    * $a = 0$
    * $b = -1 + 2\nu$
    * $c = -\nu$
* Правая часть:
    * $p1 = 0$
    * $p2 = 0$
    * $q1 = \tau$
    * $q2 = 0$

#### Фрагмент кода MATLAB:

```matlab
% Explicit Euler Scheme Coefficients
a = 0;
b = -1 + 2*nu;
c = -nu;
p1 = 0;
p2 = 0;
q1 = tau;
q2 = 0;
```

### 2. Схема Кранка-Николсона (cn)

Схема Кранка-Николсона - популярный метод второго порядка для численного решения дифференциальных уравнений.

#### Коэффициенты:

*   Левая часть:
    *   $a = -\frac{\nu}{2\alpha}$
    *   $b = \frac{-1 + \nu}{\alpha}$
    *   $c = -\frac{\nu}{2\alpha}$
*   Правая часть:
    *   $p1 = 0$
    *   $p2 = 0$
    *   $q1 = \frac{\tau}{2\alpha}$
    *   $q2 = \frac{\tau}{2\alpha}$
*   где $\alpha = 1 + \nu$

#### Фрагмент кода MATLAB:

```matlab
% Crank-Nicolson Scheme Coefficients
alpha = 1 + nu;
a = -nu/2/alpha;
b = (-1 + nu)/alpha;
c = -nu/2/alpha;
p1 = 0;
p2 = 0;
q1 = tau/2/alpha;
q2 = tau/2/alpha;
```

### 3. Компактная схема

Компактная схема разработана для обеспечения высокой точности и эффективности решения дифференциальных уравнений.

#### Коэффициенты:

*   Левая часть:
    *   $a = \frac{2(1 - 6\nu)}{\alpha}$
    *   $b = \frac{-4(5 - 6\nu)}{\alpha}$
    *   $c = \frac{-2(1 + 6\nu)}{\alpha}$
*   Правая часть:
    *   $p1 = \frac{\tau}{\alpha}$​
    *   $p2 = \frac{\tau}{\alpha}$​
    *   $q1 = \frac{10\tau}{\alpha}$
    *   $q2 = \frac{10\tau}{\alpha}$
*   где $\alpha = 4(6\nu + 5)$

#### Фрагмент кода MATLAB:

```matlab
% Compact Scheme Coefficients
alpha = 4*(6*nu + 5);
a = 2*(1 - 6*nu)/alpha;
b = -4*(5 - 6*nu)/alpha;
c = -2*(1 + 6*nu)/alpha;
p1 = tau/alpha;
p2 = tau/alpha;
q1 = 10*tau/alpha;
q2 = 10*tau/alpha;
```
